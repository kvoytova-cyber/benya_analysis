# От Петербурга до Парижа: мемуары человека Серебряного века в цифрах

Количественный анализ мемуаров Александра Бенуа «Мои воспоминания» (5 книг, ~750 тыс. токенов) 
методами DH.

## О проекте

Учебный проект по программированию. Цель — проверить, какие характеристики 
автобиографического текста меняются вместе с биографией автора, а какие остаются устойчивой 
частью его стиля.

Основной вывод: **переменные** признаки (лексика, круг лиц, география, иноязычные вставки) 
согласованно показывают «движение наружу» — от мира семьи к европейскому художественному 
авангарду; **инвариантные** признаки (части речи, длина предложения, эмоциональный фон) 
остаются стабильными.

## Что внутри

| Анализ | Метод |
| --- | --- |
| Базовая статистика корпуса | токены, леммы, TTR, длина предложения |
| Таймлайн жизни | извлечение дат регулярками |
| Частотная и характерная лексика | частоты, TF-IDF |
| Грамматический профиль | части речи (pymorphy3) |
| Иноязычные вставки | регулярки + словари служебных слов |
| Эмоциональная тональность | NRC Emotion Lexicon (рус.) |
| Граф персон и топонимика | NER (spaCy) + networkx |
| Художественный репертуар | извлечение названий в кавычках |

## Стек

Python, pandas, pymorphy3, nltk, spaCy (`ru_core_news_sm`), scikit-learn, networkx, 
matplotlib, seaborn.

## Данные

- `benua_books.csv` — корпус: 5 строк, по одной книге.
- `Russian-NRC-EmoLex.txt` — словарь эмоций NRC (русская версия).

## Ограничения

NER и лемматизация без снятия омонимии дают шум (имена правились вручную через словарь 
соответствий); детекция языка по диакритике огрублена; эмоции по NRC — тенденция, 
а не точная метрика. Подробнее — в комментариях к соответствующим разделам.
